One keyboard. Every machine.

Synbad is a free, open-source desktop client for sharing one keyboard, mouse, and clipboard across nearby computers. LAN auto-discovery, peer-to-peer config sync — no cloud account, no telemetry.

Download View on GitHub

Built on the open-source Synergy Core (GPLv2, upstream of Deskflow). Not affiliated with or endorsed by Symless.

What it does

Shared input

Move your cursor between machines like they're one display. Keyboard, mouse, and clipboard follow.

LAN auto-discovery

Peers find each other over mDNS — no manual IPs, hostnames, or accounts. Pair once, stay paired.

Config sync, peer-to-peer

Edit your screen layout on any machine; every peer converges. No central server, no cloud.

Cross-platform GUI

Native client for Linux, macOS, and Windows. Modern config UI, optional system tray.

Encrypted transport

Pairing handshake plus authenticated, encrypted peer-to-peer channels — see the security model.

Open source

MIT-licensed client; supervises the GPLv2 Core fetched on first run. No bundled binaries, no surprises.

Install

Pre-built binaries land on the GitHub releases page once tagged. Until then, build from source: 

git clone https://github.com/krazyjakee/Synbad
cd Synbad
cargo build --release --workspace

Per-platform autostart installers live in dist/: systemd user unit (Linux), launchd agent (macOS), Task Scheduler entry (Windows).
